关闭redis

关闭redis有2种方法：

  • 方法1
    在执行启动命令的终端窗口使用ctrl+c,此时第一个窗口输出

然后在另外一个终端窗口执行$ redis-cli ping,输出

说明确实已关闭

  • 方法2
    在另外一个终端窗口执行$ redis-cli shutdown,此时第一个窗口输出

然后在另外一个终端窗口执行$ redis-cli ping,输出

说明确实已关闭
